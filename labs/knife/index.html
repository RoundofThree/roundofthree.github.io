Knife - HackTheBox machine

2021-09-07 :: RoundofThree
#HackTheBox 
  • Rank: easy

Foothold⌗

  1. Nmap to find ports 22 and 80. There seems to be no exploit from Apache and OpenSSH versions.
  2. Inspect the HTML and the web request. Use nikto to scan. Found PHP version 8 dev, for which there is an exploit in ExploitDB.
  3. Get user shell using the backdoor PHP exploit.

Privilege escalation⌗

  1. sudo -l to find knife. knife can execute ruby script, so:
sudo knife exec -E 'exec "/bin/bash"'

Note that we didn’t use SSH. Another potential solution would be to bruteforce the SSH connection.
